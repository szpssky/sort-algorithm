Quick Start

Add maven dependency

<dependency>
  <groupId>com.tifosi-m</groupId>
  <artifactId>sort-tool</artifactId>
  <version>1.0</version>
</dependency>

Or Download sort-algorithm.jar

import com.tifosi.tool.sort.*;


Sort<Integer> quickSort = new QuickSort<Integer>(array);
quick.sort();
